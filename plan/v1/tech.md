# 技术栈

- 平台：iOS / Android demo，使用 Capacitor
- 前端：Vue 3 + Ionic
- 本地数据存储：SQLite（Capacitor 插件）
- 资源文件存储：应用沙盒目录 + Filesystem 插件
- 设备能力：
  - Camera
  - Local Notifications
  - Geolocation
  - Filesystem
- Summary / Mail 渲染：HTML + 内联 JavaScript + Canvas + WebView
- 服务端同步：本版不做，`token` 仅为后续预留
- Web 端：已从本版 demo 范围移除
